Для запуска сервера:
  1: cd server;
  2: yarn install;
  3. Добавить ".env файл с переменной PRIVATE_KEY(например, PRIVATE_KEY=asdfghjyuty654weqaszcxvbnm,kjlio987623qewsfdvc)"
  3: yarn start:dev;

Для запуска клиента:
  1:cd client;
  2:yarn install;
  3:ng serve;

Ссылка на документацию swagger: http://localhost:5000/docs;
